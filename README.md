# Taller 6 - Laravel con MariaDB y Bootstrap

## Objetivo

Implementar una aplicación básica en Laravel conectada a una base de datos MariaDB usando XAMPP/phpMyAdmin.

El proyecto permite gestionar registros de la tabla `proyectos` mediante vistas Blade.

## Tecnologías utilizadas

- Laravel
- PHP
- MariaDB
- XAMPP
- Bootstrap
- Blade

## Funcionalidades

- Listar proyectos.
- Crear proyectos.
- Editar proyectos.
- Mostrar datos desde la base de datos.
